# Назва проєкту
CoreDesk

## Проблема
У процесі навчання та роботи велика кількість посилань на зустрічі, навчальних матеріалів та дедлайнів розпорошена по різних сервісах та чатах. Це створює хаос, призводить до втрати фокусу та ризику пропустити важливі завдання через відсутність єдиної системи контролю.

## Рішення
Застосунок виступає централізованим хабом, який збирає всі організаційні моменти в одному місці. Він дозволяє зберігати структуру матеріалів, швидкі посилання на онлайн-зустрічі та вести персональний календар із нагадуваннями про дедлайни, що допомагає ефективно керувати часом.

## Цільовий користувач
Студенти та активні люди, які поєднують навчання з роботою чи іншими проектами та потребують інструменту для структурування свого освітнього процесу.
